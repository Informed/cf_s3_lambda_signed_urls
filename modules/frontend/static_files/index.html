What is this site

This is a demonstration site where static files, a Lambda-based API, and a private bucket are all hosted on a single domain

You can open the devtools to see that no other domain is involved, even for file uploading and signed URLs

Use this input field to upload to the private bucket

This uses S3 POST signed URLs which uploads objects to a private S3 bucket
